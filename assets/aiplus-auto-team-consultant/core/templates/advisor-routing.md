# Advisor Routing Template

Use when the current session is advising rather than executing.

```text
SESSION_ROLE=Advisor
REQUEST_SUMMARY=
DEFAULT_TIER=LIGHT
SELECTED_TIER=LIGHT | MEDIUM | HEAVY
TIER_REASON=
INVOKED_LENSES=[]
SKIPPED_LENSES_WITH_REASONS=[]
OWNER_GATE_NEEDED=YES | NO
CEO_HANDOFF_REQUIRED=YES | NO
```

Advisor rule: direct advice is preferred unless risk, ambiguity, or execution handoff justifies escalation.
