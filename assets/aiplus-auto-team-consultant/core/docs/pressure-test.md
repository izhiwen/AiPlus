# Pressure-Test

`SIMULATED_PRESSURE_TEST_ONLY`

Pressure-Test is simulated stakeholder input. It helps identify possible confusion, trust concerns, accessibility concerns, and copy or flow fixes.

It is not real user research, validation, safety approval, accessibility approval, or release approval.

Default simulated perspectives:

- Non-technical user
- Older adult user
- Low vision / large text user
- Keyboard / screen reader user
- Busy executive / time-poor user
- Privacy-anxious user
