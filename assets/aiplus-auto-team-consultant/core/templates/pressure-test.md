# Pressure-Test Template

`SIMULATED_PRESSURE_TEST_ONLY`

Pressure-Test is simulated input only. It is not real user research, validation, a study, accessibility approval, safety approval, or release approval.

```text
PRESSURE_TEST
LABEL=SIMULATED_PRESSURE_TEST_ONLY
SUBJECT=
WORKFLOW_LEVEL=LIGHT | MEDIUM | HEAVY
SIMULATED_PERSPECTIVES=[
  Non-technical user,
  Older adult user,
  Low vision / large text user,
  Keyboard / screen reader user,
  Busy executive / time-poor user,
  Privacy-anxious user
]
LIKELY_CONFUSIONS=[]
TRUST_CONCERNS=[]
ACCESSIBILITY_CONCERNS=[]
COPY_OR_FLOW_FIXES=[]
WHAT_THIS_DOES_NOT_PROVE=[
  real user validation,
  accessibility conformance,
  safety approval,
  public-release readiness
]
```
