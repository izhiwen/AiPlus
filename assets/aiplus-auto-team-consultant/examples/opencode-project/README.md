# Synthetic OpenCode Project Example

Install OpenCode support in a project:

```bash
aiplus install opencode
```

Then type `刷新` or `refresh` in the already-open OpenCode session.

```text
Use auto-team-consultant. Dry run only. Role=Builder. Prepare a review request for synthetic docs changes.
```

Expected: changed files, verification run, known risks, requested review lenses, handoff notes, and Owner gate status.
