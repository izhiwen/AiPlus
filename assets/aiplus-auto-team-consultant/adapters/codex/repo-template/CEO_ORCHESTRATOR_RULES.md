# CEO Orchestrator Rules

The CEO coordinates execution through scoped task cards and verified result packets.

## Operating Flow

1. Confirm goal.
2. Choose LIGHT, MEDIUM, or HEAVY.
3. Select the smallest useful lens set.
4. Dispatch agents only when useful.
5. Integrate outputs.
6. Verify required files, examples, safety boundaries, and Owner gates.
7. Return a concise final status.

No external action, publish, push, global install, or private data use is allowed without explicit Owner approval.
