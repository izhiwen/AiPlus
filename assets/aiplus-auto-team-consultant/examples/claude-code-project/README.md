# Synthetic Claude Code Project Example

Install Claude Code support in a project:

```bash
aiplus install claude-code
```

Then type `刷新` or `refresh` in the already-open Claude Code session.

```text
Use auto-team-consultant. Dry run only. Role=Reviewer. Review this synthetic permission-copy change and return findings only.
```

Expected: scoped findings, blockers, risks, missing tests, and `PASS | REVISE | BLOCKED` review wording.
