# Modules

## core

Runtime-neutral protocol, docs, and templates for session-local team consultation.

## adapters/codex

Codex instruction adapter. Preserves the source Codex Skill behavior and includes Codex repo-template files.

## adapters/claude-code

Project-local Claude Code skill, commands, and agents. Does not modify global config.

## adapters/opencode

Project-local OpenCode example config, commands, agents, and prompts. Does not modify global config.

## examples

Synthetic examples for shared protocol and each runtime.

## docs

Migration, compatibility, public boundary, and QA documentation.
