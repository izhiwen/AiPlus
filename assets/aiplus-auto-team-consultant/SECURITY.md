# Security And Public Safety

AiPlus Auto Team Consultant is a project-local workflow module installed by the Rust-first `aiplus` CLI. This module repo does not include package registry publishing, binary uploads, MCP server, App connector, remote service, telemetry, deployment flow, global install flow, or external account integration.

## Public Safety Disclaimer

This kit provides session-local decision-support and team-routing structure only.
It does not guarantee safety, compliance, correctness, privacy, legal readiness, product quality, or public-release readiness.
Simulated pressure-tests are not real user research.
Owner approval cannot be inferred or simulated.
The current agent remains responsible for scope control, verification, and Owner-gated actions.

## Stop For Owner Gate

Stop and request explicit Owner approval before:

- publishing or pushing to a remote
- creating a remote repository
- globally installing this module outside the project-local `aiplus install` flow
- modifying global Codex, Claude Code, OpenCode, or other agent configuration
- contacting external accounts
- deploying or uploading artifacts
- adding telemetry
- calling remote services
- adding an MCP server or App connector
- using private project data outside the active target
- exposing private data externally or uploading it
- using private data not explicitly provided or named by the Owner for this task
- exposing secrets, tokens, or private paths
- claiming compliance, safety, legal, quality, or public-release readiness
- presenting simulated pressure-tests as real user research

## Data Hygiene

Examples must remain synthetic. Templates must not include customer data, internal logs, secrets, proprietary paths, private project names, or account identifiers.
