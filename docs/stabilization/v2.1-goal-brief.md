## Goal

Evolve AiPlus from v0.5.1 stable baseline to v2.1 with focused improvements in:
1. Compact Reminder resume/capsule integration
2. Memory doctor deep scanning
3. Profile bundle schema validation
4. Installed binary upgrade path

## Scope Included

- Compact resume reads `context-capsule.json`
- Decision ledger extraction (replace stub)
- Memory doctor deep scan (redaction, stale, conflict)
- Profile bundle identity/schema validation
- Installed binary upgrade verification
- Documentation updates

## Scope Excluded (Same as v0.5.1)

- AppModules product modules
- Cloud sync
- Vector database
- True daemon/launchd
- Full transcript auto-learning
- Automatic approved skills
- Payment/voice
- Telemetry
- Global config edits
- crates.io/npm/Homebrew publish

## Acceptance Criteria

All v2.1 tasks must:
- Pass `cargo fmt/clippy/test`
- Pass Runtime QA smoke checks
- Preserve public/private/secret boundaries
- Not introduce telemetry or global config edits
- Include unit tests for new logic
