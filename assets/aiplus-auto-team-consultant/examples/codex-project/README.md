# Synthetic Codex Project Example

Install Codex support in a project:

```bash
aiplus install codex
```

Then type `刷新` or `refresh` in the already-open Codex session.

```text
Use auto-team-consultant. Dry run only. Role=Advisor. Review this synthetic onboarding prompt for calendar access. Return Consultant Packet only. Do not edit files.
```

Expected: `WORKFLOW_LEVEL=LIGHT | MEDIUM`, selected lenses, skipped lenses with reasons, risks, and recommended next action.
